# Majestic Translator

> Оверлей-переводчик чата для [Majestic RP](https://majestic-rp.ru/) и других мультиплеерных серверов GTA V в реальном времени. Считывает чат игры через экранный OCR, переводит русский → польский (или любую пару, которую укажешь), а ответ на польском кладёт переводом на русском прямо в буфер обмена.

**Русский** · [Polski](README.pl.md) · [English](README.md)

## Зачем

Сервер русскоязычный, ты говоришь на польском. Или наоборот. Или учишь язык. В клиенте перевода нет, сторонние переводчики чат не читают, копи-пастить каждое сообщение из игры рушит RP-погружение.

Majestic Translator — это маленькое окошко поверх игры. Тихо распознаёт область чата, гонит каждую новую строку через Google Translate и показывает перевод в порядке поступления. Обратное направление — через поле ввода и буфер обмена.

## Возможности

- 🎯 **OCR по выделенной области** — откалибровал один раз, навёл на панель чата; работает на любом разрешении.
- ⚡ **Skip по разности кадров** — когда чат не меняется, OCR не запускается, приложение в idle ~0% CPU.
- 🧠 **Умная склейка сообщений** — длинные перенесённые реплики собираются обратно в одну; UI-надписи и собственное окно маскируются перед распознаванием.
- 🚀 **Параллельный неблокирующий перевод** — медленные Google-запросы не тормозят захват чата; результаты выходят строго в порядке появления в чате.
- 🌍 **3 языка интерфейса** — русский, польский, английский.
- 🎨 **3 темы** — Majestic (тёмная + золото), Dark (графит), Light (белая).
- 🪶 **Нативное безрамочное окно** — drag через системный API, без рваной самописной логики.
- 🔓 **Без API-ключей** — публичный endpoint Google Translate (тот же, что использует «Перевести страницу» в Chrome).

## Быстрый старт

### Вариант A — установщик (Windows)

1. Скачай `MajesticTranslator-Setup.exe` из раздела [Releases](../../releases).
2. Запусти, всё по умолчанию, готово.
3. Запусти из меню «Пуск».
4. На первом запуске обведи мышью область чата в игре.

### Вариант B — из исходников

Нужен **Python 3.11+** и интернет (модели OCR скачаются один раз, ~150 МБ).

```powershell
git clone https://github.com/<ваш-юзер>/majestic-translator.git
cd majestic-translator

python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt

python main.py
```

При первом запуске на экране появится полупрозрачный оверлей — обведи мышкой область чата, отпусти. Координаты сохраняются в `config.json` и используются при последующих запусках.

## Использование

### Чтение чата игры

Просто играй. Раз в секунду окно распознаёт область чата, отбирает новые русские строки, переводит на польский и добавляет в ленту (новые снизу).

- **Оригинал** показан выцветшим под каждым переводом.
- В ленте хранятся последние 40 сообщений.
- Статусная строка показывает состояние OCR/перевода.

### Отправка польский → русский

1. Напиши польское сообщение в нижнем поле ввода.
2. Нажми **Ctrl+Enter** (или кнопку **ПЕРЕВЕСТИ И СКОПИРОВАТЬ**).
3. Русский перевод копируется в буфер.
4. В игре: **T** → открыть чат, **Ctrl+V** → вставить, **Enter** → отправить.

### Переключение языка и темы

В нижней статусной строке два выпадающих списка:

- **Язык**: язык интерфейса (Русский / Polski / English). Направление перевода чата при этом не меняется — оно всё равно RU ↔ PL.
- **Тема**: Majestic (тёмная + золото), Dark (нейтральный графит), Light (белая).

Оба выбора мгновенно сохраняются в `config.json`.

### Перекалибровка области чата

Иконка **⌖** в заголовке окна. Снова появится оверлей — обведи новую область.

## Конфигурация

`config.json` создаётся рядом с экзешником при первом запуске:

| Ключ | Описание | По умолчанию |
|---|---|---|
| `chat_region` | Область OCR в пикселях экрана | (выставляется при калибровке) |
| `ocr_interval_ms` | Интервал опроса чата. Ниже = отзывчивее, больше CPU | `1000` |
| `overlay_pos` / `overlay_size` | Геометрия окна | сохраняется между запусками |
| `source_lang` / `target_lang` | Направление перевода чата | `ru` → `pl` |
| `input_lang` / `input_target_lang` | Направление перевода поля ответа | `pl` → `ru` |
| `language` | Язык интерфейса: `ru` \| `pl` \| `en` | `ru` |
| `theme` | Тема: `majestic` \| `dark` \| `light` | `majestic` |

## Как это работает

```
                ┌─────────────────────────┐
                │  Majestic RP / GTA V    │
                │  ┌───────────────────┐  │
  область чата  │  │     чат игры      │  │
   ═════════════╪══╪═══════════════════╪══╪═══
                │  │  [HH:MM] реплика… │  │
                │  └───────────────────┘  │
                └─────────────┬───────────┘
                              │ mss.grab()
                              ▼
                    ┌─────────────────┐
                    │ skip по разнице │── без изменений ──▶ выход
                    └────────┬────────┘
                             │ изменилось
                             ▼
                    ┌─────────────────┐
                    │ PaddleOCR (ru)  │── строки + боксы
                    │ mobile det+rec  │
                    └────────┬────────┘
                             ▼
                    ┌─────────────────┐
                    │ склейка по «-»  │── строки чата
                    └────────┬────────┘
                             ▼
                    ┌─────────────────┐
                    │ дедуп [HH:MM]   │── новые строки
                    └────────┬────────┘
                             │ seq-номер
                             ▼
                    ┌─────────────────┐
                    │ пул переводов   │── параллельно × 4
                    │ Google /gtx     │   упорядоченный emit
                    └────────┬────────┘
                             ▼
                       лента в окне
```

Ключевое:
- **Модель детекции**: `PP-OCRv5_mobile_det` (~10 МБ, ~150 мс/цикл на CPU).
- **Модель распознавания**: `eslav_PP-OCRv5_mobile_rec` (восточно-славянская — русский / украинский / белорусский).
- **Перевод**: прямой вызов `translate.googleapis.com/translate_a/single?client=gtx`. Тот же endpoint, что Chrome использует для перевода веб-страниц. Без API-ключа.
- **Потоки**: OCR-цикл никогда не ждёт перевод — медленные Google-запросы не приостанавливают захват чата. `ThreadPoolExecutor` гоняет переводы параллельно; буфер с seq-номерами восстанавливает исходный порядок.

## Сборка из исходников

Нужно:
- Python 3.11 или 3.12
- ~5 ГБ свободного места под артефакты сборки
- (Опционально) [Inno Setup 6](https://jrsoftware.org/isdl.php) для генерации установщика

```powershell
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt pyinstaller

# 1. Собрать папку с распакованным приложением
pyinstaller build/MajesticTranslator.spec --noconfirm

# 2. (Опционально) Завернуть в установщик Windows
"%ProgramFiles(x86)%\Inno Setup 6\ISCC.exe" installer\installer.iss
```

Результат:
- `dist/MajesticTranslator/` — папка с приложением, ~700 МБ распакованной
- `installer/output/MajesticTranslator-Setup.exe` — один файл-установщик, ~250 МБ

## Приватность / что уходит в сеть

- **Модели OCR** скачиваются один раз с официального зеркала PaddleOCR (~150 МБ, кэш в `%USERPROFILE%\.paddlex\`).
- **Переводы** идут на публичный endpoint Google по HTTPS. Каждая распознанная строка чата отправляется как query-параметр. Никакой телеметрии и аналитики. Исходник в `app/translator.py:23`, если хочешь убедиться.
- **Больше никуда ничего не уходит.**

## Ограничения

- **Античит**: Majestic RP — это RAGE Multiplayer с кастомным античитом. Это приложение только **читает** экран и подделывает буфер обмена — в игру не инжектит и её не модифицирует. Но политики на других RP-серверах могут отличаться; проверь правила перед использованием.
- **Точность OCR** не идеальна на стилизованных шрифтах чата. Иногда коряво распознаются ники и тэги фракций (`Фракция` → `Фрakция` и т.п.). Сам текст сообщений обычно нормально.
- **Лимиты Google**: при очень большом потоке (~> 30 строк/мин стабильно) публичный endpoint Google может начать резать. Снизь интервал OCR или подключи платный бэкенд.
- **Один монитор сначала**: работает на нескольких мониторах, но оверлей калибровки появляется только на основном.

## Вклад

Issues и PR приветствуются. Кодовая база небольшая (~1000 строк), структура простая:

```
app/
├── config.py        # сохранение JSON-конфига
├── i18n.py          # переводы UI (RU/PL/EN)
├── themes.py        # три палитры Qt-стилей
├── ocr.py           # mss → PaddleOCR → склейка → дедуп
├── translator.py    # прямой клиент Google /gtx + защита ников/RP-команд
├── workers.py       # неблокирующий параллельный pipeline (Qt-сигналы)
├── region_picker.py # полупрозрачный оверлей для калибровки
└── overlay.py       # безрамочное главное окно PySide6
main.py              # точка входа: загрузка конфига, выбор региона, показ
```
